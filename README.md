This HTML document is a comprehensive, self-contained Employee ID Card Generator web application. It allows users to create professional dual‑side ID cards with customizable designs, data input methods, QR codes, and print‑ready outputs.

Key Features:

Dual Input Modes: Enter employee data manually (single entry) or in bulk via Excel upload (.xlsx/.csv) or tab‑separated pasted text. Supports up to eight fields: Employee ID, Name, Designation, CNIC, Board ID, Contact, Date of Joining, and Home Address.

Rich Visual Designs: Seven pre‑built card themes (BISE Official, Glass, Neon Edge, Minimal, Vintage, Corporate, Dark Tech), each with distinct colors, fonts, and borders.

Orientation: Choose between landscape or portrait card layouts.

Media Customization: Upload a company logo, an authority signature (optionally overlaid on the employee photo), and employee photos (single or batch – matched by filename against Employee ID).

QR Code Generation: Each card back includes a dynamic QR code encoding employee ID, name, role, organization, and institution.

Real‑time Preview: A live preview section reflects changes to design, orientation, text, and images immediately.

Print & Cut Utilities: Dedicated print button, toggleable cut guides (dashed borders + corner marks) and a mirror mode (horizontal flip) for specialized printing needs.

Responsive & Self‑contained: Works offline (after initial load) with local data processing; uses XLSX and QRCode libraries via CDN.

Workflow: Configure organization details, select design/orientation, upload assets, input employee data (single or bulk), click “Generate ID Cards” – the tool renders each card pair (front/back) in a grid, ready for printing or further review.
